# Keep It Cheesy

![picture](/readme-shot.png)

Keep It Cheesy is a fictional gourmet cheese store, showcasing their production process and product selection. It enables users to send inquiries and custom orders using a contact form. The website was created using JavaScript, Sass, HTML, and GitHub.

* [Link to deployed version](http://keepitcheesy.minabui.com/)
* [Read more on Mina Bui's portfolio site](https://www.minabui.com/brochure)

### Built with

* [HTML](https://html.spec.whatwg.org/)
* [Sass](https://sass-lang.com/)
* [Javascript](https://www.javascript.com/)
* [Google Fonts](https://fonts.google.com/)
